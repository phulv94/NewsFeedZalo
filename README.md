## ðŸ›  Architecture

### MVVM
- **Model**: Represents Diary and Video data (id, title, description, thumbnail, etc.).
- **ViewModel**: Exposes processed data to Views.
- **View**: SwiftUI components (`DiaryTabView`, `VideoTabView`).

### Data Manager (Singleton)
